# 运行故事钩子评分

## 目标

按页检查 `story_graph` 的翻页欲望、信息增量、情绪牵引和视觉记忆点。

## 必读

- `创作者技法库/质量门禁/钩子评分模型.md`
- `模板/平台页钩子评分模板.md`
- 当前项目 `故事核心.json`

## 评分项

- `hook_strength`
- `information_gain`
- `emotional_pull`
- `visual_memory`
- `next_page_pull`
- `clarity`

## 硬失败

- `hook_strength < 3`
- `next_page_pull < 3`
- `information_gain = 0`
- 连续两页没有新问题。
- 中段没有反转或升级。
- 结尾没有 final_echo。

## 输出

- 分数表。
- 硬失败列表。
- 修复动作。
