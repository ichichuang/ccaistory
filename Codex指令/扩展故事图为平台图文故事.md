# 扩展故事图为平台图文故事

## 必读

- `创作者技法库/技法总览.md`
- `创作者技法库/组合矩阵/故事类型到技法矩阵.md`
- `结构规范/技法增强故事图结构规范.md`
- `结构规范/故事钩子强化结构规范.md`
- 当前项目 `故事核心.json`

## 执行

1. 读取创作者技法库。
2. 选择技法组合矩阵。
3. 为每页写入 `applied_skills`。
4. 为每页生成 hook 字段。
5. 为每个 story_graph node 补齐 `typed_hook`、`typed_narration`、`final_hook_sentence`、`child_note_candidates`、`source_illustration_focus`、`platform_page_layout`、`reader_retention_goal`、`visual_memory_point`、`suspense_type`、`payoff_target`。
6. 为每条 edge 补齐 `cliffhanger_from`、`answer_in_next`、`unresolved_question`、`payoff_deadline`、`question_from_previous`、`answer_in_current`、`new_question_created`、`clue_carried_forward`、`tension_delta`。
7. 运行钩子评分。
8. 评分不通过则停止并写修复动作。

## 禁止

- 不生成 prompt。
- 不生成图片。
- 不生成发布材料。
- 不使用作者姓名作为风格指令。
