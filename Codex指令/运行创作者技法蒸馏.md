# 运行创作者技法蒸馏

## 目标

从合法公开资料或用户提供的一手素材中蒸馏无作者名的高层叙事技法。

## 输入

- 蒸馏对象或主题。
- 合法来源 URL 或本地素材路径。
- 目标故事类型。

## 步骤

1. 读取 `创作者技法库/来源白名单.md`。
2. 读取 `创作者技法库/版权与引用边界.md`。
3. 用 `模板/技法来源记录模板.md` 记录来源短摘要。
4. 用 `模板/创作者技法蒸馏模板.md` 写入蒸馏字段。
5. 删除作者名、作品可识别细节、原文句式和画面风格。
6. 只输出可作用于 `story_graph` 的技法。

## 禁止

- 不复制作者原文。
- 不复制作者画风。
- 不生成作者风格 prompt。
- 不保存完整作品、付费内容全文或大段访谈。

## 验收

- 每条技法都有 `validation_questions` 和 `repair_actions`。
- 每条技法都有 `forbidden_copy_boundary`。
- 每条技法都能写入 `story_graph_fields_affected`。
