# 接入nvwa技能

## 目标

把 `nvwa.skill` 或本地 `huashu-nuwa` 蒸馏能力接入创作者技法库，不复制原技能，不破坏原路径。

## 必读

- `创作者技法库/nvwa接入/接入说明.md`
- `创作者技法库/nvwa接入/nvwa技能适配规范.md`
- `模板/nvwa技能接入模板.md`

## 步骤

1. 搜索项目内和本地技能目录是否存在 `nvwa.skill`。
2. 找到时只读取接口说明、目录结构和可接入方式。
3. 找不到时登记待人工补充路径。
4. 输出适配层字段。
5. 不复制原 skill 正文。

## 禁止

- 不覆盖原 `nvwa.skill`。
- 不把蒸馏结果直接转为作者风格 prompt。
- 不生成图片、任务清单或发布材料。
