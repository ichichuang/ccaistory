# 运行图文分镜与图读测试

## 执行目标

读取故事重写、分集计划和页面文字脚本；补齐图文分镜字段；执行图读测试；失败时停止在视觉设定前。

## 禁止

- 不生成图片。
- 不创建发布内容。
- 不修改原始摘录或参考包。
- 不使用完整任务清单绕过试产批准。

## 输出

- 更新对应项目 Markdown 文件。
- 更新 [[总控/总索引]] 与 [[总控/操作日志]]。
- 报告验证命令和阻断项。

## 故事核心状态机要求

执行前必须读取 `故事核心.json`。新故事必须先创建故事核心，不得直接创建独立分镜事实文件。分镜、干读、总览只作为派生视图生成。试产结果必须回填到故事核心的 `quality_gates` 和 `prompt_plan`。
