# 术语表

| 术语 | 定义 |
|---|---|
| 故事核心 | 当前故事的唯一事实源，记录状态、结构、门禁和派生视图索引。 |
| story_graph | 平台图文故事的结构化页面图，包含节点、边、钩子、情绪递进和回收点。 |
| 平台图文故事 | 面向移动端阅读的图文主线生产模式。 |
| 派生视图 | 从故事核心生成的分镜、图读测试、视觉项目总清单等工作文件。 |
| 状态机 | 约束故事从输入到发布的阶段顺序和允许动作。 |
| 门禁 | 阻断不合格内容进入下游阶段的验收条件。 |
| 创作者技法 | 从公开来源抽象出的叙事机制，不包含作者风格模仿或可识别表达。 |
| 视觉项目总清单 | 记录角色、场景、道具、版式和资产依赖的生产清单。 |
| 提示词编译 | 将故事核心和视觉清单转换为可执行提示词材料。 |
| 出图任务清单 | 外部图像生产前使用的任务列表，不等同于已产图片。 |
| 资产验收 | 对生成资产进行尺寸、语义、连续性、安全性和可用性检查。 |
| 发布材料 | 发布页排版、文案和上线前检查材料。 |
| 数据回收 | 发布后记录表现数据和规则校准建议。 |
