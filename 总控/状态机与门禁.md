# 状态机与门禁

## 核心原则

每个项目必须由 `故事核心.json.machine_state` 控制阶段推进。任何派生文件只能登记为 `derived_views` 或 `visual_pipeline`，不得成为第二事实源。

## 标准字段

- `current_state`：当前已完成或阻断的状态。
- `gate_result`：`pending`、`passed`、`rework_required` 或 `blocked`。
- `next_allowed_action`：唯一下一入口。
- `blocked_actions`：当前必须禁止的动作。
- `blocker`：阻断原因和证据。

## 全局 blocked_actions

未通过上游门禁时，必须阻断：

- generate_images
- generate_prompt_package
- full_generation
- publish
- asset_accept
- create_publish_package

## 视觉执行链门禁

- 视觉项目总清单不存在：阻断视觉资产本体。
- 视觉资产本体不存在：阻断 Prompt 编译。
- compiled_prompt 不存在：阻断语义 Lint、任务清单和外部出图。
- semantic_lint 不是 passed：阻断任务清单和外部出图。
- pilot_task_list 不存在：阻断外部出图。
- execution_telemetry 不存在：阻断资产级验收。
- actual_prompt_sent_to_external_tool 不存在：阻断 accepted。
- 资产级细粒度验收不存在：阻断 accepted。
- reference_dependencies 不满足：阻断 accepted。

## R00 专项门禁

R00 只负责纸张、笔触、短线索和材料质感。R00 不得出现人物、火柴人、完整场景、道具集合、符号散点表、UI 线框、长正文或制作说明。R00 未 accepted 时，所有依赖 R00 的 R01、R02、P01 和 S 页资产保持 blocked。

## 失败回退

- story_core 字段缺失：回退故事核心状态机技能。
- 页数或连续性失败：回退故事图扩展与页数审查。
- 图读失败：回退平台图文故事编排。
- 本体失败：回退视觉项目总清单或视觉资产本体。
- prompt 失败：回退 Prompt 编译。
- Lint 失败：回退 Prompt 编译或本体。
- 遥测缺失：回退图像执行遥测；无法补录则 blocked。
- 验收失败：回退返修队列，不能直接 accepted。
