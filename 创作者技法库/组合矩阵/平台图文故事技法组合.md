# 平台图文故事技法组合

状态：活动

## 默认组合

| 层级 | 技法 | 作用 |
| --- | --- | --- |
| 必选 | 平台完读率技能 | 控制首屏、三页留存、中段升级和结尾分享 |
| 必选 | 翻页钩子技能 | 每页制造下一页理由 |
| 必选 | 情绪牵引技能 | 让读者关心角色 |
| 必选 | 世界感压缩技能 | 用少量视觉锚点建立空间 |
| 可选 | 恐怖递进技能 | 适合怪谈和悬疑 |
| 可选 | 命运闭环技能 | 强化结尾复读价值 |
| 可选 | 儿童视角奇观技能 | 适合儿童儿童视角图文 |

## 页序要求

- 第 1 页：`first_page_hook` 必须成立。
- 第 3 页前：规则、目标、危险方向必须出现。
- 中段：必须有 `mid_story_turn`。
- 终段：必须有 `pre_final_acceleration`。
- 最后一页：必须有 `final_share_trigger` 或 `final_echo`。

## 硬失败

- 第一页像设定页。
- 三页内没有规则或危险方向。
- 中段没有升级。
- 每页没有 `final_hook_sentence`。
