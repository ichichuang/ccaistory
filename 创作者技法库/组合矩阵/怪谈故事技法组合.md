# 怪谈故事技法组合

状态：活动

## 默认组合

| 层级 | 技法 | 作用 |
| --- | --- | --- |
| 必选 | 日常异常化技能 | 用普通物件建立怪谈入口 |
| 必选 | 规则怪谈技能 | 压缩危险规则 |
| 必选 | 恐怖递进技能 | 从不对劲升级到无法忽视 |
| 必选 | 翻页钩子技能 | 保证页间拉力 |
| 可选 | 误导与反证技能 | 制造合理误读 |
| 可选 | 命运闭环技能 | 结尾回收开头 |
| 可选 | 志怪转折技能 | 结尾重构现实解释 |

## story_graph 要求

- 每页写入 `applied_skills`。
- 至少 1 条规则写入 `rule_text` 或 `rule_pressure`。
- 中段必须有 `contradiction` 或规则升级。
- 结尾必须有 `final_echo` 或 `rule_payoff`。

## 硬失败

- 怪谈只靠氛围词。
- 没有可复述规则。
- 日常物件没有回收。
- 连续两页没有新问题。
