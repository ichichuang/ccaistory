# 儿童化改编技法组合

状态：活动

## 默认组合

| 层级 | 技法 | 作用 |
| --- | --- | --- |
| 必选 | 儿童化改编技能 | 删除成人风险，只保留抽象机制 |
| 必选 | 儿童视角奇观技能 | 用儿童能理解的物件和误会表达危险 |
| 必选 | 日常异常化技能 | 把危险落在普通道具上 |
| 必选 | 命运闭环技能 | 让结尾回收开头信号 |
| 可选 | 规则怪谈技能 | 适合一句规则驱动 |
| 可选 | 情绪牵引技能 | 强化主角目标和补救 |
| 可选 | 世界感压缩技能 | 适合少地点故事 |

## 必删内容

- 血腥。
- 尸体。
- 枪械。
- 性暗示。
- 成人犯罪细节。
- 复杂法律/警方线。
- 大段心理独白。

## 硬失败

- 保留原作可识别句式或桥段。
- 成人风险元素仍在。
- 儿童无法通过画面理解危险。
- 改编后仍依赖原作人物关系。
