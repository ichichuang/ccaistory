# 钩子评分模型

状态：活动  
作用范围：story_graph 钩子强化、平台图文故事编排、图读测试、人工完整阅读

## 单页评分

每页 0 到 5 分：

| 维度 | 5 分标准 | 0 分标准 |
| --- | --- | --- |
| `hook_strength` | 页尾有清晰问题、危险或反常证据 | 页尾只是陈述 |
| `information_gain` | 本页提供新证据、新规则或新风险 | 本页没有新信息 |
| `emotional_pull` | 角色想要、害怕、内疚或误会清楚 | 读者不关心角色 |
| `visual_memory` | 有可记住的物件、位置、颜色或动作 | 画面无记忆点 |
| `next_page_pull` | 读者知道下一页要回答什么 | 下一页无期待 |
| `clarity` | 问题、证据和动作一眼能读懂 | 读者不知道发生什么 |

## 硬失败

- `hook_strength < 3`
- `next_page_pull < 3`
- `information_gain = 0`
- 连续两页没有新问题。
- 中段没有反转或升级。
- 结尾没有 `final_echo`。

## 修复动作

- 弱钩子：补 `next_page_question` 和 `final_hook_sentence`。
- 无信息增量：补新证据、新误读、新规则压力或新矛盾。
- 情绪弱：补 `character_want`、`character_fear` 或 `emotional_stake`。
- 视觉记忆弱：补 `visual_memory_point` 和重复地标。
- 下一页拉力弱：补 `answer_delay` 和 `payoff_deadline`。
- 清晰度弱：删设定堆叠，改为可见动作和短规则。
