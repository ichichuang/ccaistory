# 故事钩子强化验收清单

状态：活动

## 必检项

- [ ] 每页有 `typed_hook`。
- [ ] 每页有 `next_page_question`。
- [ ] 每页有 `answer_delay`。
- [ ] 每页有 `final_hook_sentence`。
- [ ] 每页有 `reader_retention_goal`。
- [ ] 每条边有问题-回答-新问题链路。
- [ ] 中段有反转或升级。
- [ ] 结尾有开头回扣。

## 硬失败

- `hook_strength < 3`。
- `next_page_pull < 3`。
- `information_gain = 0`。
- 连续两页没有新问题。
- 中段没有反转或升级。
- 结尾没有 final_echo。
