# 创作者技法蒸馏验收清单

状态：活动

## 必检项

- [ ] 来源来自白名单。
- [ ] 每条来源只保存短摘要。
- [ ] 已填写 `source_summary`。
- [ ] 已填写 `observed_pattern`。
- [ ] 已填写 `abstract_principle`。
- [ ] 已填写 `usable_skill`。
- [ ] 已填写 `story_graph_fields_affected`。
- [ ] 已填写 `validation_questions`。
- [ ] 已填写 `repair_actions`。
- [ ] 技法可作用于 `story_graph`。

## 硬失败

- 复制作者原文。
- 复制作者画风。
- 保存完整作品或大段版权文本。
- 蒸馏结果只能描述作者风格相似度。
- 无法转写为 `story_graph` 字段。
