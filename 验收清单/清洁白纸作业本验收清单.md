# 清洁白纸作业本验收清单

状态：活动

## 必查项

- [ ] 干净白色或极浅象牙白横线作业本纸。
- [ ] 淡蓝横线和红色边线稳定。
- [ ] 禁止黄旧、复古、档案、羊皮纸、茶渍、破边。

## 硬失败

- [ ] 长篇复杂故事少于 10 页。
- [ ] 每页没有 typed_narration。
- [ ] 每页没有下一页钩子。
- [ ] 图内手写承担长正文。
- [ ] 纸张黄旧或像档案纸。
- [ ] 画面是设定页不是故事页。
- [ ] 主线页只有标题。
- [ ] 试产通过后直接生成完整包。
- [ ] source_illustration 和 platform_story_page 混为一体。
- [ ] 过期整页图文模式被当作完整发布模式。
