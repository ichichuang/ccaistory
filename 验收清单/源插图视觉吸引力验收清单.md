# 源插图视觉吸引力验收清单

状态：活动

## 必查项

- [ ] 第一眼可判断画面用途。
- [ ] 有明确视觉中心。
- [ ] 元素有主次层级。
- [ ] 儿童笔触可读，不是成人设计图标。
- [ ] 轻微怪谈气息存在但不压倒画面可用性。
- [ ] 可作为后续 reference asset 或故事页视觉基准。

## R00 必查项

- [ ] 只测试清洁白纸作业本、短笔迹、红圈、箭头、问号、划线、笔触、铅笔压痕和橡皮痕。
- [ ] 不画人物、不画火柴人、不画完整场景。
- [ ] 不画隧道、值班屋、红灯、电铃、小旗、门、铁路切口或道具集合。
- [ ] 不做符号散点表、素材表、流程图、UI 规划图或制作说明。

## 硬失败

- [ ] 火柴人。
- [ ] 符号散点表。
- [ ] 无视觉中心。
- [ ] 元素过多且无层级。
- [ ] 页面像草稿测试纸。
- [ ] 页面像道具清单。
- [ ] 页面像 UI 线框。
- [ ] 页面像制作说明。
- [ ] 画面只有约束，没有审美。
- [ ] 儿童感等于乱画。
- [ ] 锚图不可作为后续 reference asset。
- [ ] 第一眼没有可用性。
