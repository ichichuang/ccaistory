# 源插图视觉吸引力锁

对象：source_illustration 与 reference anchor。

目标：画面必须干净、有秩序、有儿童笔触、有轻微怪谈气息、有视觉层次，可作为后续图像统一参考。它不是专业插画，也不是乱画废纸。

## 正向目标

- 第一眼能看出画面用途。
- 有明确视觉中心。
- 元素有主次层级，不平均散落。
- 儿童感来自不齐、压线、轻微涂改和材料痕迹，不来自混乱。
- 怪谈气息轻微、克制、可读。
- 画面可作为后续 reference asset 统一纸张、笔触、角色、道具或场景。

## 硬失败

- 火柴人。
- 符号散点表。
- 无视觉中心。
- 元素过多且无层级。
- 页面像草稿测试纸。
- 页面像道具清单。
- 页面像 UI 线框。
- 页面像制作说明。
- 画面只有约束，没有审美。
- 儿童感等于乱画。
- 锚图不可作为后续 reference asset。
- 第一眼没有可用性。

## R00 附加口径

R00 只负责清洁白纸作业本、横线、红边、纸张颗粒、儿童短线索字迹、红圈、箭头、问号、划线笔触、铅笔压痕、圆珠笔/彩铅/蜡笔痕和橡皮擦痕。

R00 不负责人物、道具清单、场景清单、故事情节、隧道、值班屋、信号灯完整设定、平台页排版或图文故事吸引力测试。
