# 平台发布页排版模板

对象：platform_story_page
推荐尺寸：1080x1920，9:16 竖屏。

```yaml
page_id: {page_id}
page_number: {page_number}
layout_ratio: 9:16
typed_hook: {8-24字}
typed_narration: {40-160字}
source_illustration_id: {source_illustration_id}
child_note_layer:
  - text: {2-8字短线索}
    mark: {arrow|circle|question|underline|crossout}
final_hook_sentence: {8-32字}
next_page_question: {下一页必须回答的问题}
reader_retention_goal: {为什么读者会翻页}
```

主叙事使用排版文字，不交给图像模型手写。
