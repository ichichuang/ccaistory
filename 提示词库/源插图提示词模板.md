# 源插图提示词模板

对象：source_illustration
比例：默认 3:4，除非故事核心另有项目指定。

```text
source_illustration_id: {id}
story_page_id: {page_id}
scene_action: {本页可见事件推进}
emotion: {情绪变化}
visual_memory_point: {读者会记住的视觉点}
style: childlike hand-drawn source illustration on clean white ruled school notebook paper
paper: clean white or very pale ivory ruled school notebook paper, faint blue horizontal lines, stable red margin line
child_note_layer: {2-8 字短线索，最多 3 条，可含箭头、红圈、问号或划掉}
visual_quality: see 源插图视觉吸引力锁；must have visual center, hierarchy, controlled child marks, and reusable reference value
forbidden_text: no long paragraph, no full narration, no adult calligraphy, no polished title layout
negative: see 负面约束；no stick figure, no scattered symbol sheet, no prop catalog, no UI layout, no production notes
```

源插图只负责画面和情绪，不负责平台发布排版。
