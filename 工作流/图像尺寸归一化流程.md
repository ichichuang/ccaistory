# 图像尺寸归一化流程

提示词不能保证精确像素。下载图必须验证尺寸；非规范图只能 padding 或 crop 到规范尺寸，禁止拉伸。归一化必须保留作业本红线、蓝线和文字安全区。最终故事输出只能包含规范尺寸。

## 固定门禁

- 图文分镜与图读测试通过 → 视觉设定。
- 视觉项目总清单通过 → 提示词生成。
- 试产任务清单巡检通过 → 三页公开试产。
- 三页公开试产通过 → 完整出图任务清单。
- 每两张相邻主线页通过连续性检查 → 下一批。
- 全部主线通过图读测试和人工完整阅读 → 发布编排。
