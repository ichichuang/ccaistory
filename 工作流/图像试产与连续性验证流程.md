# 图像试产与连续性验证流程

流程: 批准文字故事 → 批准图文分镜和图读测试 → 批准视觉项目总清单 → 生成 R00 → R01 → R02 → C00 → S01 → S02 → 试产 QA → 人工批准后每批最多两张相邻主线页 → 每批连续性 QA → dense text 二次编辑 → 尺寸归一化 → 候选收口。

完整任务清单在试产人工批准前不得激活。试产阶段唯一允许的资产 ID 是 R00、R01、R02、C00、S01、S02。

## 固定门禁

- 图文分镜与图读测试通过 → 视觉设定。
- 视觉项目总清单通过 → 提示词生成。
- 试产任务清单巡检通过 → 三页公开试产。
- 三页公开试产通过 → 完整出图任务清单。
- 每两张相邻主线页通过连续性检查 → 下一批。
- 全部主线通过图读测试和人工完整阅读 → 发布编排。
