# 视觉一致性表规范

视觉一致性表按 image_id 记录角色比例、道具状态、地点状态、天气、纸张、调色、参考依赖和相邻页关系。每批出图后更新。

## 通用硬门禁

- 缺少必填字段时阻断下游。
- 资产 ID 类别必须属于 Rxx、Sxx、Axx、Cxx、Txx、Fxx。
- 所有资产类别必须使用 3:4 portrait 和 1536x2048 px。
- 不得混用横版、不同尺寸或未登记候选资产。
