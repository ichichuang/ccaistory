# 视觉资产本体结构规范

## 目的
防止视觉资产职责混淆。所有外部出图 prompt 必须先由本体定义。

所有视觉资产规格必须包含非空 `paper_policy` 与 `style_policy`。`S_SOURCE_ILLUSTRATION` 不允许省略这两个字段；缺失时必须硬阻断 Prompt 编译。

## 顶层结构

```json
{
  "asset_id": "",
  "asset_type": "",
  "asset_scope": "",
  "story_context": "",
  "allowed_content": [],
  "forbidden_content": [],
  "visual_center": "",
  "density_range": "",
  "composition_mode": "",
  "text_policy": {},
  "paper_policy": {},
  "style_policy": {},
  "reference_dependencies": [],
  "acceptance_questions": [],
  "repair_policy": {}
}
```

## asset_type

- `R00_PAPER_MARK_ANCHOR`
- `R01_CHARACTER_ANCHOR`
- `R02_SCENE_PROP_ANCHOR`
- `P01_PLATFORM_LAYOUT_SAMPLE`
- `S_SOURCE_ILLUSTRATION`

## R00_PAPER_MARK_ANCHOR
职责：纸张、蓝线、红边、纸张颗粒、儿童短线索笔迹、红圈、箭头、问号、划线、铅笔压痕、橡皮痕。
禁止：人物、火柴人、完整场景、道具清单、隧道、电铃、值班屋、小旗、铁路、符号散点表、UI 线框、制作说明、长正文。

## R01_CHARACTER_ANCHOR
职责：角色比例、儿童画法人物、服饰与轮廓合同。禁止完整剧情和场景大图。

## R02_SCENE_PROP_ANCHOR
职责：场景空间关系、关键异常物、道具比例。禁止主线事件和结局。

## P01_PLATFORM_LAYOUT_SAMPLE
职责：9:16 平台页版式。禁止伪装成完整故事图。

## S_SOURCE_ILLUSTRATION
职责：单页故事画面、角色动作、情绪、source_illustration_focus、visual_memory_point、child_note_layer。禁止长正文和设定表。

`paper_policy` 固定约束清洁白色手账/作业本纸面，可有轻微铅笔压痕、橡皮痕、小红圈、小箭头、问号和短儿童笔记；禁止泛黄旧纸、脏纸、档案纸、羊皮纸、烧边和大面积空白试卷。

`style_policy` 固定约束儿童安全恐怖手账源插图、9-12 岁儿童画影响但可读、略歪但受控的铅笔/圆珠笔线条、克制红色标记点缀；禁止精修数字概念图、通用 AI 恐怖构图、假印刷手写字和过度电影光照。
