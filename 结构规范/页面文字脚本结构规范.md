# 页面文字脚本结构规范

页面文字脚本必须列出标题、主叙述、页边注解、物件标签、箭头目标、圈注目标、划掉修正、文字区、总字数和 text_render_mode。主线默认 28 到 60 个中文字符，4 到 8 个文字元素。

## 通用硬门禁

- 缺少必填字段时阻断下游。
- 资产 ID 类别必须属于 Rxx、Sxx、Axx、Cxx、Txx、Fxx。
- 所有资产类别必须使用 3:4 portrait 和 1536x2048 px。
- 不得混用横版、不同尺寸或未登记候选资产。
