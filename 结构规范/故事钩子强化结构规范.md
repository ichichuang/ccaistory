# 故事钩子强化结构规范

状态：活动  
唯一事实源：故事核心.json

## 目的

定义翻页钩子、下一页问题、回答延迟和评分门禁。

## 单页钩子字段

```json
{
  "typed_hook": "",
  "next_page_question": "",
  "answer_delay": "",
  "final_hook_sentence": "",
  "reader_retention_goal": "",
  "hook_score": {
    "hook_strength": 0,
    "information_gain": 0,
    "emotional_pull": 0,
    "visual_memory": 0,
    "next_page_pull": 0,
    "clarity": 0
  }
}
```

## 边关系字段

```json
{
  "question_from_previous": "",
  "answer_in_current": "",
  "new_question_created": "",
  "clue_carried_forward": "",
  "payoff_deadline": "",
  "tension_delta": "up"
}
```

## 评分模型

每项 0 到 5 分：

- `hook_strength`
- `information_gain`
- `emotional_pull`
- `visual_memory`
- `next_page_pull`
- `clarity`

## 硬失败

- `hook_strength < 3`
- `next_page_pull < 3`
- `information_gain = 0`
- 连续两页没有新问题。
- 中段没有反转或升级。
- 结尾没有 `final_echo`。

## 修复顺序

1. 修复信息增量。
2. 修复页尾问题。
3. 修复下一页回答。
4. 修复中段升级。
5. 修复结尾回扣。
