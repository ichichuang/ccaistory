# 技法增强故事图结构规范

状态：活动  
唯一事实源：故事核心.json

## 目的

定义创作者技法库作用于 `story_graph` 后必须出现的节点、边和评分字段。

## node 新增字段

- `applied_skills`：本页使用的技法数组。
- `typed_hook`：本页开头或页内钩子。
- `reader_retention_goal`：读者继续阅读的目标。
- `next_page_question`：本页制造的下一页问题。
- `answer_delay`：问题延迟回答策略。
- `suspense_type`：悬疑/怪谈压力类型。
- `emotional_turn`：本页情绪转折。
- `visual_memory_point`：本页视觉记忆点。
- `rule_pressure`：规则压力或违反风险。
- `contradiction`：与合理解释冲突的证据。
- `final_hook_sentence`：页尾钩子句。
- `payoff_target`：本页埋下或回收的目标。
- `technique_notes`：技法应用备注。

## edge 新增字段

- `question_from_previous`：上一页留下的问题。
- `answer_in_current`：本页如何回答。
- `new_question_created`：本页新增问题。
- `clue_carried_forward`：跨页延续线索。
- `payoff_deadline`：最迟回收页。
- `tension_delta`：张力变化，取值 `up`、`flat`、`down`。

## 项目级字段

`technique_profile` 必须包含：

```json
{
  "mode": "platform_graphic_story",
  "skill_library": "creator_technique_library",
  "selected_skill_set": [],
  "hook_strength_required": 3,
  "next_page_pull_required": 3,
  "information_gain_required": 1,
  "full_story_target_pages": 12,
  "full_generation_blocked_until": "skill_enhanced_story_graph_pass"
}
```

## 硬失败

- 任一正式主线页缺少 `applied_skills`。
- 任一正式主线页缺少 `final_hook_sentence`。
- 连续两页没有 `new_question_created`。
- 中段没有 `tension_delta: up` 或反转。
- 结尾没有回收 `payoff_target`。
