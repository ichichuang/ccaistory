# 画布与版式结构规范

画布结构必须包含 exact aspect ratio、orientation、target dimensions、safe margins、title zone、image zone、main caption zone、margin annotation zones。

## 通用硬门禁

- 缺少必填字段时阻断下游。
- 资产 ID 类别必须属于 Rxx、Sxx、Axx、Cxx、Txx、Fxx。
- 所有资产类别必须使用 3:4 portrait 和 1536x2048 px。
- 不得混用横版、不同尺寸或未登记候选资产。
