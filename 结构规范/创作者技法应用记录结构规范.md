# 创作者技法应用记录结构规范

状态：活动  
唯一事实源：故事核心.json

## 目的

记录每个项目如何选择、应用和验证创作者技法库。

## 记录字段

```json
{
  "technique_application_record": {
    "story_id": "",
    "project_id": "",
    "source": "creator_technique_library",
    "selected_skill_set": [],
    "combination_matrix": "",
    "nodes_checked": 0,
    "edges_checked": 0,
    "hard_failures": [],
    "repair_actions": [],
    "gate_result": "pass"
  }
}
```

## 必填说明

- `selected_skill_set`：项目已选择的技法。
- `combination_matrix`：使用的组合矩阵路径。
- `nodes_checked`：已检查节点数。
- `edges_checked`：已检查边数。
- `hard_failures`：硬失败列表。
- `repair_actions`：修复动作。
- `gate_result`：`pass`、`conditional_pass`、`fail`、`blocked`。

## 硬失败

- 技法选择未写入项目。
- 技法应用没有节点级证据。
- 钩子评分未记录。
- 禁止模仿规则未检查。
