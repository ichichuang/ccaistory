# 资产级细粒度验收结构规范

## 通用结构
asset_id、asset_type、questions、hard_failures、result。

## R00 必问
1. 是否有人物？必须否。
2. 是否有火柴人？必须否。
3. 是否有完整场景？必须否。
4. 是否有道具集合？必须否。
5. 是否是符号散点表？必须否。
6. 是否像 UI/流程图？必须否。
7. 是否像乱涂垃圾纸？必须否。
8. 是否纸张清洁白色？必须是。
9. 是否没有泛黄旧纸？必须是。
10. 是否有可控儿童笔迹样本？必须是。
11. 是否短线索数量合理？必须是。
12. 是否没有长正文？必须是。
13. 是否有视觉中心或秩序？必须是。
14. 是否可作为后续 reference asset？必须是。

任一不满足：R00 不得 accepted，不得继续 R01/R02，必须回到 Prompt 编译器修复。
