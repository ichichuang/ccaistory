# 平台发布页模板

模板对象：`platform_story_page`
唯一事实源：故事核心.json

```yaml
object_type: platform_story_page
story_id: {{story_id}}
project_id: {{project_id}}
source_field_path: {{story_core_field_path}}
derived_view_path: {{derived_view_path}}
gate_result: pending

platform_requirements:
  source_illustration_ratio: "3:4"
  platform_page_ratio: "9:16"
  typed_narration_required: true
  child_note_max_chars_each: 8
  full_generation_blocked_until: platform_story_graph_expanded

page:
  page_id: {{page_id}}
  page_number: {{page_number}}
  page_role: {{page_role}}
  typed_hook: {{8_to_24_chars}}
  typed_narration: {{40_to_160_chars}}
  final_hook_sentence: {{8_to_32_chars}}
  source_illustration_id: {{source_illustration_id}}
  child_note_layer:
    - text: {{2_to_8_chars}}
      mark: arrow_or_circle_or_question
  image_focus: {{image_focus}}
  next_page_question: {{question}}
  emotional_turn: {{emotion_change}}
  visual_memory_point: {{visual_memory_point}}
  reader_retention_goal: {{why_continue}}

hard_failures:
  - long_complex_story_under_10_pages
  - missing_typed_narration
  - missing_next_page_hook
  - long_handwritten_narration_inside_image
  - yellow_or_aged_paper
  - setting_page_instead_of_story_page
  - title_only_main_page
  - full_package_generated_from_pilot
```
