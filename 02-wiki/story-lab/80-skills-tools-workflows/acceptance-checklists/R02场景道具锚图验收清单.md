# R02 场景道具锚图验收清单 / R02 Scene & Prop Anchor Acceptance Checklist

## Purpose / 目的
验收 R02 场景/道具锚图：确认场景与关键道具的一致性与可复用性，作为场景连续性的 accepted 依赖来源。

## Required Inputs / 必需输入
- R02 候选图、对应 Scene 卡（`props`、`continuity_constraints`）、人工 `image_review_form`。
- 已 accepted 的 R00（纸张笔触锚图）。

## Pass Criteria / 通过标准
- [ ] 场景与 Scene 卡 canonical 描述一致（空间结构、关键道具、连续性约束）。
- [ ] 关键道具形态稳定、可跨页复用。
- [ ] 不越界承载完整角色系统（角色由 R01 提供）。
- [ ] 儿童安全；可作为场景参考复用。

## Fail Criteria（硬失败 / Hard Failures）
- [ ] 携带完整剧情 / 完整角色系统 / 符号散点表。
- [ ] 道具与连续性约束冲突；场景与 Scene 卡冲突。
- [ ] 不安全内容；作者姓名风格指令。

## Blocking Conditions / 阻断条件
- R00 未 accepted → 阻断。
- 未做人工图像复核 → 阻断。

## Required Evidence / 必需证据
- `image_review_form` + `asset_qa_result`（`50-agent-work`）。

## Output Decision / 输出决策
- `accepted` → ReferenceAsset（`visual_role: scene-anchor`）；`rework` → RepairNote；`rejected` → 90-archive。

## Target Layer / 落地层
- `50-agent-work/story-lab/qa-results/`；accepted 结论回填 `02-wiki/story-lab/reference-assets/`。

## Related Templates & Runtime / 关联模板与命令
- 模板：`Scene`、`ReferenceAsset`。命令：`qa-asset`。
