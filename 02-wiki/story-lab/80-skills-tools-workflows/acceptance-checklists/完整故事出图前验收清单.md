# 完整故事出图前验收清单 / Pre-Illustration Final Acceptance Checklist

## Purpose / 目的
在整篇故事进入图像生产前的总门禁：确认所有 canonical 卡齐全、上游门禁全过、执行包可编译，避免半成品进入出图。

## Required Inputs / 必需输入
- StoryProject、稳定 story_graph、全部必需 Character/Scene/VisualStyle/PromptRecipe 卡、平台图文页编排、图读测试结果。

## Pass Criteria / 通过标准
- [ ] 故事核心、页数合理性、钩子强化、连贯性门禁全部 passed。
- [ ] 每个必需场景都有对应 Scene/Character/VisualStyle/PromptRecipe canonical 卡。
- [ ] 图读测试通过（仅靠图文可读懂）。
- [ ] 待出图的 ImageExecutionPackage 均可 `compile-asset` + `lint-asset` 通过、`status: ready`。

## Fail Criteria / 失败标准
- [ ] 缺任一必需 canonical 卡。
- [ ] 任一上游门禁未过；图读测试失败。
- [ ] 执行包编译/lint 不通过。

## Blocking Conditions / 阻断条件
- 任一上游门禁未过或卡缺失 → 阻断，不得进入 G 出图。

## Required Evidence / 必需证据
- 上游门禁结果（runtime `can-run` / `check-graph` 输出）、图读测试记录，落 `50-agent-work/story-lab/qa-results/`。

## Output Decision / 输出决策
- `pass` → 进入图像链路（E→F→G）；`fail` → 返修最近上游。

## Target Layer / 落地层
- `50-agent-work/story-lab/qa-results/`。

## Related Templates & Runtime / 关联模板与命令
- 模板：`StoryProject`、`ImageExecutionPackage`。命令：`can-run`、`check-graph`。
