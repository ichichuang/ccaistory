# 人工完整阅读清单 / Manual Full-Read Checklist

## Purpose / 目的
在发布前由人工完整阅读已装配的成品，捕捉错序、遮挡、断裂与发布风险——成品发布前的最后一道人工门禁。

## Required Inputs / 必需输入
- 成品包计划（StoryProject Final Package 区）、全部 accepted ReferenceAsset、平台图文页编排。

## Pass Criteria / 通过标准
- [ ] 整篇仅靠图 + 文即可读懂主要事件与情绪。
- [ ] 页序正确，翻页钩子连贯。
- [ ] 无图文遮挡、无叙事断裂。
- [ ] 儿童安全；无版权/合规风险。

## Fail Criteria / 失败标准
- [ ] 错序、遮挡、断裂、关键信息缺失。
- [ ] 出现未 accepted 或 rejected 资产。
- [ ] 不安全或越权内容。

## Blocking Conditions / 阻断条件
- 任一理解性失败 → 阻断发布，回 K 装配或对应上游。

## Required Evidence / 必需证据
- 复核人签收（reviewer + 时间）+ 阅读笔记，落 `50-agent-work/story-lab/qa-results/`。

## Output Decision / 输出决策
- `pass` → 置 StoryProject `publishing_readiness_status: ready`；`fail` → 阻断并返修。

## Target Layer / 落地层
- `50-agent-work/story-lab/qa-results/`；最终发布决策回填 `02-wiki` StoryProject。

## Related Templates & Runtime / 关联模板与命令
- 模板：`StoryProject`。命令：`status`（复核 machine_state）。
